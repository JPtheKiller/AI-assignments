Código para compilação:

javac DTree.java

Código para execução:

java DTree

Ficheiros .csv têm de estar na mesma pasta do que o ficheiro compilado.
Os ficheiros de teste são ficheiros .csv iguais aos fornecidos, mas sem a coluna ID e Class,
com os valores pretendidos para teste em cada linha. São fornecidos dois ficheiros de exemplo.

Programa foi compilado e executado em Linux Min 18.3 Cinnamon 64-bit.
